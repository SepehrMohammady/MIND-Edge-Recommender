۲.۲ دیتاست xMIND و NLLB-3.3B

xMIND نسخه چندزبانه MIND است — عناوین و چکیده مقالات با مدل ترجمه NLLB-3.3B (No Language Left Behind) به 14 زبان ترجمه شده‌اند.

NLLB-3.3B چیست؟

ویژگیتوضیح
نامNo Language Left Behind — Meta (Facebook) 2022
پارامترها۳.۳ میلیارد (اینجا "3.3B" می‌آید)
معماریTransformer encoder-decoder برای ترجمه چندزبانه
زبان‌های پشتیبانی‌شده۲۰۰+ زبان (پرمنبع و کم‌منبع)
داده آموزشی NLLBویکی‌پدیا + Flores + CCMatrix + داده‌های parallel
💡 چرا کیفیت ترجمه متفاوت است؟
تمام 14 زبان xMIND یکسان ترجمه شده‌اند، اما کیفیت ترجمه به اینکه NLLB برای هر زبان چقدر داده parallel دارد بستگی دارد:

زبان‌های پرمنبع (zho، jpn): NLLB میلیاردها جملات برای این زبان‌ها دارد → ترجمه عالی
زبان‌های کم‌منبع (kat، tam): NLLB فقط میلیون‌ها جملات دارد → ترجمه ضعیف‌تر

می‌توانم زبان دیگری (مثل Italian یا Farsi) اضافه کنم؟

بله! اگر می‌خواهید نسخه xMIND را برای Italian یا Farsi گسترش دهید، می‌توانید NLLB-3.3B را مستقیم استفاده کنید:

Python
from transformers import pipeline

# NLLB از کدهای زبان (eng_Latn / ita_Latn / pes_Arab) و آرگومان‌های
# src_lang و tgt_lang استفاده می‌کند — نه task مثل translation_en_to_xx
trans = pipeline("translation", model="facebook/nllb-200-3.3B",
                 src_lang="eng_Latn", tgt_lang="ita_Latn")
title_en = "Breaking: France wins World Cup"
print("Italian:", trans(title_en)[0]["translation_text"])

# فارسی: tgt_lang را به pes_Arab تغییر بده
trans_fa = pipeline("translation", model="facebook/nllb-200-3.3B",
                    src_lang="eng_Latn", tgt_lang="pes_Arab")
print("Farsi:", trans_fa(title_en)[0]["translation_text"])

اما: xMIND یک دیتاست منتشرشده و ثابت است. این 14 زبان برای تنوع و بنچ‌مارک‌سازی انتخاب شدند. اگر می‌خواهید Italian یا Farsi به پروژه خود اضافه کنید، باید:

۱
NLLB-3.3B را برای ترجمه MIND انگلیسی استفاده کنید
۲
embedding‌های entity و relation را برای زبان جدید تولید کنید
۳
مدل‌های استاد و دانشجو را برای زبان جدید آموزش دهید
⚠️ نکته ارزیابی — Cross-lingual Transfer
xMIND فقط متن مقالات را ترجمه می‌کند. رفتار کاربران (کلیک‌ها) همان داده انگلیسی MIND است. پس ارزیابی cross-lingual transfer است، نه رفتار بومی هر زبان.
