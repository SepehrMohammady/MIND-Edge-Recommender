۱.۱ مسئله را بشناسیم

هر روز میلیون‌ها مقاله خبری منتشر می‌شود. کاربر وقت ندارد همه را بخواند. یک Recommender System (سیستم توصیه‌گر) مقاله‌هایی را نشان می‌دهد که بیشترین احتمال کلیک دارند.

کاربر ← [تاریخچه کلیک‌ها] ↓ سیستم توصیه‌گر ↓ رتبه‌بندی مقالات جدید → نمایش بهترین‌ها
💡 مثال ساده
تاریخچه: فوتبال، اقتصاد، فناوری
مقالات جدید: [آب‌وهوا، پیروزی تیم ملی، بورس، هوش مصنوعی، آشپزی]
توصیه: پیروزی تیم ملی ✅ | بورس ✅ | هوش مصنوعی ✅

راه‌حل: حذف جدول لغات با Byte-level Encoding

به جای نگهداری جدول لغات برای هر زبان، ما از 256 بایت UTF-8 به عنوان واژگان استفاده می‌کنیم. یک مدل → همه زبان‌ها!

Python
# این کد را در notebook پروژه اجرا کنید
text_en = "Breaking News"
text_fa = "اخبار مهم"
text_ja = "最新ニュース"

for text in [text_en, text_fa, text_ja]:
    byte_ids = list(text.encode("utf-8"))
    print(f"'{text}' → {len(byte_ids)} bytes → {byte_ids[:8]}...")
