۷.۲ فرمت ONNX چیست؟

ONNX (Open Neural Network Exchange) یک فرمت استاندارد باز برای مدل‌های هوش مصنوعی است. مدل PyTorch صادر می‌شود و با ONNX Runtime روی هر سخت‌افزاری اجرا می‌شود.

PyTorch model ↓ torch.onnx.export() content_encoder.onnx ↓ ┌────────────────────────────────┐ │ ONNX Runtime │ ├──────────────┬─────────────────┤ │ Windows/Linux│ aarch64 (Pi 5) │ │ CPUExecProv │ CPUExecutionProv│ └──────────────┴─────────────────┘

۷.۴ اجرا با ONNX Runtime (Python)

Python — OnnxRuntime
import onnxruntime as ort, numpy as np, json

# بارگذاری state و model
state = json.load(open("artifacts/edgeml_model_state_v1.json"))
sess  = ort.InferenceSession("artifacts/content_encoder.onnx",
         providers=["CPUExecutionProvider"])

max_len = state["contentEncoder"]["inputBytes"]

def encode_title(title: str) -> np.ndarray:
    """عنوان را به embedding تبدیل کن."""
    b = [x + 1 for x in title.encode("utf-8")][:max_len]   # +1 چون 0 = pad
    b += [0] * (max_len - len(b))   # padding
    x = np.array([b], dtype=np.int64)
    return sess.run(None, {"title_bytes": x})[0]

# آزمون با چند زبان
titles = {
    "en": "Sports news from the world",
    "zho": "世界体育新闻",
    "tur": "Dünya spor haberleri",
}
for lang, title in titles.items():
    emb = encode_title(title)
    print(f"[{lang}] shape={emb.shape}, norm={np.linalg.norm(emb):.3f}")

۷.۶ مسیر استقرار

۱

RTX 5070 Blackwell / Windows / Linux

onnxruntime با CPUExecutionProvider — مستقیم اجرا می‌شود. GPU با CUDAExecutionProvider.

۲

Raspberry Pi 5 (aarch64)

نصب onnxruntime نسخه aarch64 و همان کد Python — بدون تغییر. برای اندازه‌گیری واقعی انرژی، INA219 سنسور اضافه کنید.

۳

STM32H7 (MCU)

ONNX → X-CUBE-AI برای INT8. فایل .onnx را import کنید و X-CUBE-AI کد C بهینه‌شده برای Cortex-M7 می‌سازد.

⚠️ محدودیت Binary روی MCU
هیچ toolchain کاملی برای PyTorch → Binary → STM32H7 وجود ندارد. X-CUBE-AI نسخه 1-bit را فقط از Larq/TensorFlow می‌خواند. اعداد Binary برای MCU در این پروژه تحلیلی (Horowitz proxy) هستند، نه اندازه‌گیری واقعی.

Python — End-to-End Demo
import onnxruntime as ort, numpy as np, json, time
from numpy.linalg import norm

state = json.load(open("artifacts/edgeml_model_state_v1.json"))
sess  = ort.InferenceSession("artifacts/content_encoder.onnx",
         providers=["CPUExecutionProvider"])
L = state["contentEncoder"]["inputBytes"]

def embed(text):
    b = [x + 1 for x in text.encode("utf-8")][:L]   # +1: 0 = pad
    b += [0] * (L - len(b))
    return sess.run(None, {"title_bytes": np.array([b], dtype=np.int64)})[0][0]

# مقاله با ترجمه‌های مختلف
article = {
    "en": "World Cup: France wins championship",
    "zho": "世界杯：法国赢得冠军",
    "tur": "Dünya Kupası: Fransa şampiyonluğu kazandı",
    "jpn": "ワールドカップ：フランスが優勝",
}

embs = {lang: embed(title) for lang, title in article.items()}

print("شباهت cosine با نسخه انگلیسی:")
en = embs["en"]
for lang, emb in embs.items():
    sim = np.dot(en, emb) / (norm(en) * norm(emb))
    print(f"  [{lang}] cosine = {sim:.4f}")
